# Citation update

Replace the repository-root `CITATION.cff` with the file in this folder and add
`CITATION.bib` to the repository root.

The CFF file powers GitHub's **Cite this repository** interface. The BibTeX file
is a convenient copy-ready entry for LaTeX/BibTeX users.

After the book chapter is formally published, add a complete `preferred-citation`
block to `CITATION.cff` with the editors, book title, publisher, page range, year,
and DOI or stable URL.
