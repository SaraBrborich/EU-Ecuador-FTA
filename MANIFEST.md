# Package manifest

This package contains:

- cleaned Stata 17 source code;
- an import-side shift-share instrument using 2014 shares and 2016 tariff changes;
- figure-generation code for the five figures in the chapter;
- estimation and table-construction code for the online appendix;
- detailed data and workflow documentation;
- the final online appendix;
- the revised working chapter supplied on July 7, 2026;
- the official EU-Andean Protocol of Accession and tariff schedules.

It contains no administrative microdata or firm-level derived datasets.
